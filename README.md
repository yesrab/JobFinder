# MERN JobFinder app

- Frontend : React
- Backend : NodeJs server based on Express
- Database : NoSQL MongoDB

##### Environment  variables :
    GEOAPI=your GeolocationApi
    PORT=your server port number
    JWT_SUPER_SEACRETE=your jwt super seacreat
    DB=your mongo db URI
##### To build :
    npm install
    npm run dev
    npm run dev:server
